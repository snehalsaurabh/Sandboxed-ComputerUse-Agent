Operating Model

What this app is showing

  • Each run emits structured thought, action, and observation events.
  • Execution remains bounded by a tool policy and step budget.
  • Local-first is the default, but provider selection is modular.
  • Browser automation is intentionally deferred to a later, isolated phase.

How To Use

Minimal workflow

  1. Enter a short, concrete goal.
  2. Choose a provider mode.
  3. Start the run and watch the timeline update.
  4. Stop the run if you want to cut it short.
